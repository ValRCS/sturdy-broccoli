External libraries and Fetch API

External libraries

There are two types of external libraries in web development: Namely CSS libraries and Javascript libraries

Some libraries include both CSS and Javascript

External libraries are used to make our lives easier

Library means a code written by someone else(or many people) meant to be reused

External libraries are usually hosted on a server and we can include them in our code by using a link tag

There are also frameworks which provide a more rigid structure to your code

The difference between a full framework and a library is not completely fixed

CSS libraries

Let's look at some examples of CSS libraries

Most popular CSS library is still Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development. It contains CSS- and (optionally) JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Home page for Bootstrap is GetBootstrap

Note: full Bootstrap also includes some JS for some effects
