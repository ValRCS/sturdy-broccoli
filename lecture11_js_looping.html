Loops in Javascript

Purpose of loops:

  • Execute a block of code a number of times
  • Repeat a block of code while a condition is true

While loops

While loops are used to repeat a block of code while a condition is true

Syntax for while loop is:

        while (condition) {
            // code block to be executed
        }

Example: Print the numbers 0 to 9

For loops

For loops are used to repeat a block of code a number of times

Syntax for for loop is:

for (initialization; condition; increment) {
    code block to be executed
}

Example: Print the numbers 0 to 9
