Javascript functions

What is a function?

Functions are a way to group code together to perform a specific task. They are reusable and can be called as many times as needed.

Function declaration

Functions are declared using the function keyword followed by the name of the function, followed by a set of parentheses. The parentheses can contain parameters, which are variables that are used to store values that are passed into the function. The code block is enclosed in curly braces.

Function declaration example

        
            function myFunction() {
                // code to be executed
            }

DOM Events

What is a DOM event?

DOM events are actions that happen on the DOM tree. They are triggered by the user or the browser. DOM events are a way to interact with the user and the browser.

DOM events are a way to interact with the user and the browser.

DOM stands for Document Object Model. The DOM is a tree-like structure that represents the HTML document. The DOM is used to connect web pages to scripts or programming languages. The DOM is a programming interface for HTML and XML documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects. That way, programming languages can connect to the page.

DOM events are sent to notify code of interesting things that have taken place. Each event is represented by an object which is based on the Event interface, and may have additional custom fields and/or functions used to get additional information about what happened.

DOM event types

There are many different types of DOM events. The most common DOM events are:

  • load
  • unload
  • click
  • mouseover
  • mouseout
  • keydown
  • keyup
  • keypress
  • submit
  • change
  • focus
  • blur
